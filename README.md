# 프로젝트 소개   
#### 프로젝트명 : Pet Place (P.P)
#### 사업자는 반려동물에 관한 업체(병원, 미용, 카페)를 등록하고, 유저가 검색, 필터링 할 수 있는 웹페이지 (위치기반정보제공 서비스)

## 💡 구현기능

### 1. 유저의 위치기반 반려동물 동반 시설 정보 및 추천 (map)
### 2. 방문 시설에 대한 리뷰 작성
### 3. 필터를 사용한 조회
### 4. 유저, 사업자 모드가 분리되어 사업자의 경우 업체등록 가능
### 5. 채팅문의와 후기 등록시 알림 기능
